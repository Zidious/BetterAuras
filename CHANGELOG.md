- Stack count font size and position control
- Live preview panel
- Profile system with per-character support
- Instant application of changes
- Core addon functionality
- Configuration UI with real-time preview
- Profile management system
- Slash command interface (`/ba`)
- Settings integration with WoW's addon settings
